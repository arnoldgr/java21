# Oracle Certified Professional Java SE 21 Developer Study Guide - Source Code

Welcome to the source code repository for the **Oracle Certified Professional: Java SE 21 Developer** study guide.

This repository is designed to support students preparing for the **OCP Java SE 21 certification** by providing practical, chapter-by-chapter Java examples that align with the key concepts covered in the study guide.

The goal of this codebase is to help you move beyond theory by exploring working examples, experimenting with concepts, and building confidence as you study for the exam.

---

## About This Repository

This repository contains Java source code organized according to the major topics in the **Oracle Certified Professional Java SE 21 Developer** study guide.

As you progress through the study material, you can use this codebase to:

- Review practical examples for each chapter
- Run and test Java code as you learn
- Explore concepts hands-on
- Compare your own solutions with structured examples
- Build a strong understanding of exam topics through practice

This repository is intended to be used as a **learning companion** while studying for the certification.

---

## How Students Should Use This Codebase

To get the most value from this repository:

1. Study the theory for a chapter in your study guide.
2. Open the matching chapter package in this repository.
3. Read through the example code carefully.
4. Run the examples in IntelliJ IDEA (or your preferred Java IDE).
5. Modify the code and experiment with different outcomes.
6. Create your own practice classes for revision.
7. Revisit completed chapters when revising for the exam.

### Recommended Learning Approach

For each chapter:

- Read the concept in the study guide
- Open the corresponding chapter folder/package
- Run the code examples
- Try changing values, methods, conditions, or logic
- Observe the output
- Create your own version of the example
- Take note of tricky exam behaviors and edge cases

This is especially important for the OCP exam, where small syntax and behavior details matter.

## Chapter Navigation Guide

The source code is organized by chapter so that students can easily follow along with the study guide.

### Chapter 1: Building Blocks
Focuses on the foundations of Java, including:
- Java class structure
- `main()` method
- packages and imports
- variables
- scope
- basic syntax rules

### Chapter 2: Operators
Covers:
- arithmetic operators
- assignment operators
- comparison operators
- logical operators
- increment/decrement
- operator precedence

### Chapter 3: Making Decisions
Covers Java control flow:
- `if` statements
- `switch`
- loops (`for`, `while`, `do-while`)
- `break`
- `continue`
- nested logic

### Chapter 4: Core APIs
Introduces commonly tested APIs such as:
- `String`
- `StringBuilder`
- arrays
- basic utility methods
- object handling

### Chapter 5: Methods
Focuses on:
- method declarations
- parameters
- return types
- method overloading
- varargs
- access modifiers

### Chapter 6: Class Design Patterns
Covers:
- constructors
- encapsulation
- immutability
- inheritance basics
- object-oriented design principles

### Chapter 7: Beyond Classes
Explores:
- abstract classes
- interfaces
- enums
- records
- sealed classes
- nested classes

### Chapter 8: Lambdas and Functional Interfaces
Focuses on:
- lambda expressions
- method references
- functional interfaces
- built-in functional interfaces
- functional programming basics

### Chapter 9: Collections and Generics
Covers:
- `List`
- `Set`
- `Map`
- `Queue`
- generics
- wildcards
- collection utilities

### Chapter 10: Streams
Focuses on:
- stream creation
- intermediate operations
- terminal operations
- `Optional`
- collectors
- stream pipelines

### Chapter 11: Exceptions and Localization
Covers:
- checked vs unchecked exceptions
- `try-catch-finally`
- try-with-resources
- custom exceptions
- localization
- internationalization

### Chapter 12: Modules
Introduces the Java Platform Module System (JPMS):
- module declarations
- exports
- requires
- modular structure
- encapsulation at module level

### Chapter 13: Concurrency
Covers:
- threads
- executors
- synchronization
- concurrent collections
- thread safety
- parallel streams

### Chapter 14: I/O
Focuses on:
- file handling
- `Path`
- `Files`
- NIO.2
- reading and writing files
- serialization
- file system operations
